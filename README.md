# JD Download Telegram Bot

A powerful Telegram bot that interfaces with JDownloader to download files from various sources and upload them to Telegram.

## Features

- ✅ Download from HTTP, HTTPS, FTP, magnet links, and torrents
- ✅ Real-time progress tracking with visual progress bar
- ✅ Automatic file upload to Telegram upon completion
- ✅ Multiple user support with authorization
- ✅ File size limits configurable
- ✅ Health checks for container orchestration
- ✅ Docker and Render.com ready
- ✅ Automatic reconnection and error recovery
- ✅ Download management (cancel, pause, resume)
- ✅ Detailed logging and monitoring
